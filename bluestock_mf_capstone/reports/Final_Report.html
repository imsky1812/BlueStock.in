5. Fund Performance Analytics (D4)

5.1 Methodology — Key Formulas

# CAGR — ALWAYS use trading days (252/year), never calendar days CAGR = (NAV_end / NAV_start)^(252 / n_trading_days) - 1 # Sharpe Ratio (annualised daily) Sharpe = (mean(r - r_f) / std(r - r_f)) * sqrt(252) where r_f = 6.5% / 252 (daily risk-free rate) # Beta vs. NIFTY 50 Beta = Cov(r_fund, r_bench) / Var(r_bench) # Jensen's Alpha (annualised) Alpha = r_fund_ann - [r_f_ann + Beta * (r_bench_ann - r_f_ann)] # Max Drawdown MDD = min( (NAV_t - max(NAV_0..NAV_t)) / max(NAV_0..NAV_t) )

Only 12 out of 40 schemes (30%) generated positive Jensen's Alpha vs. NIFTY 50. 8. Monte Carlo Simulation — 5-Year Projection (B3)

8.1 Methodology

We model NAV growth using Geometric Brownian Motion (GBM), a standard financial model assuming log-normally distributed returns:

S_t = S_0 · exp[(μ - ½σ²)·t + σ·√t·Z] where: S_0 = last known NAV μ = mean daily return (computed from 5-year history) σ = daily std deviation Z ~ N(0, 1) random shocks (1,000 simulated paths) t = 1,260 trading days (5 years × 252)

9. Markowitz Efficient Frontier (B4)

9.1 Methodology

Portfolio optimisation across 5 equity funds using Modern Portfolio Theory (MPT). Two optimal portfolios were identified:

  • Minimum Variance Portfolio (MVP): Minimises σ_p subject to Σw=1, w≥0
  • Maximum Sharpe Portfolio (MSP): Maximises (r_p − r_f) / σ_p
Portfolio metrics (annualised): r_p = wᵀ·μ (portfolio return) σ_p = √(wᵀ·Σ·w) (portfolio volatility) Sharpe = (r_p − r_f) / σ_p where r_f = 6.5%
